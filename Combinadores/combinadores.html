Agrupamento de seletores

podemos aplicar as mesmas regras css para varios seletores diferentes. Para isso podemos separar os seletores utilizando vírgula (,)

Assim, não precisa ficar copiando e colando os mesmos codigos para tpdps, nós podemos fazer um só bloco de CSS e reutilizar as mesmas regras
