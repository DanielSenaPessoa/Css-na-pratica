Cores

Existem várias formas de definirmos cores utilizando o CSS. Elas são:

  • Cores pré-definidas
  • Paravra-chave "currentcolor"
  • Hexadecimal
  • Hexadecimal com transparencia
  • RGB
  • RGBA
  • HSL
  • HSLA

HSL

hsl(hue,saturation, lightness)
  • hue (matiz): é o grau de cores (de 0 a 306 graus)
    • 0 ou 360 é vermelho
    • 120 é verde
    • 240 é azul
  • Saturation (Saturação): Valor percentual
    • 0% um tom de conza
    • 100% cor total
  • lightness (luminosidade): luminosidade da cor
    • 0% é preto
    • 100% é branco
